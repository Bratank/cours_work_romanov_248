# cours_work_romanov_248

# Обучение физически информированных нейронных сетей (PINN)

Репозиторий содержит код для решения обратной задачи статистической физики — восстановления матрицы взаимодействий в двумерной модели Изинга.

## Требования
Для установки зависимостей выполните:
`pip install -r requirements.txt`

## Запуск
Откройте файл `код_курсовая_работа_Романов_Д_А_БПМИ248.ipynb` и выполните ячейки последовательно. Тетрадка запускается end-to-end.
